Project Documentation

This documentation page provides an overview of the project, its architecture, and how to use the provider factories.

Provider Factory Usage

The project exposes a simple abstraction for LLM and embedding providers via the model_providers package.

from model_providers import get_llm_provider, get_embedding_provider

llm_cfg = get_llm_provider()
llm_cfg.provider.send_message(...)

emb_cfg = get_embedding_provider()
emb_cfg.provider.embed(["text to embed"])
